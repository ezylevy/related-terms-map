How to use the graph

Features & Controls

🔍
Search
Type any word in the search box — matching nodes stay visible, others dim. An exact match auto-selects and centers the node.
🖱️
Click a node
Select any node to see its description, category, and connections. Use the → Points to and ← Points here tabs to explore outgoing and incoming links separately.
↔️
Directed edges
Arrows show the direction of each relationship. When two words point to each other, the edges curve apart so both are visible.
🗺️
Path Finder
Choose a From and To word — the graph finds the shortest directed path between them.
🎨
Category filters
Click a category button to show or hide all its nodes and edges. Each category has its own color.
📊
Statistics panel
Shows a bar chart of all categories. Click any category row to drill down and see all its words.
📜
Navigation history
A breadcrumb trail tracks every node you visit. Use ← → to go back and forward.
🔎
Zoom & minimap
Use + / − buttons or scroll to zoom. Toggle the minimap (▦) for a bird's-eye overview.
🇬🇧 🇮🇱
Switch language
Use the English / עברית toggle at the top to switch between the two graph versions.
